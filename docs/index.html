🚀 Setup in 3 steps

From zero to coding on Telegram in under 2 minutes

1

Install tgcc

One command, that's it.

npm install -g @lucashca/tgcc
2

Create a Telegram Bot

Open @BotFather on Telegram, send /newbot, and copy the token.

3

Run the wizard

Paste the token, choose a directory, send the verification code. Done! 🎉

tgcc init
